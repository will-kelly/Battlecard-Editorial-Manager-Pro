# Battlecard Editorial Manager Pro

A structured, stage-gated system for developing competitive battlecards
from concept to publish.

## What This Includes

-   Expanded GPT configuration
-   Stage-gated workflow (v2)
-   Notion database schema for pipeline tracking
-   Claude Skill configuration
-   Governance, versioning, and review structure

## Workflow Overview

1.  Intake and scoping
2.  Claims mapping and sourcing
3.  Outline development
4.  Drafting v1
5.  Review and revision
6.  Publish and enablement

Each stage enforces exit criteria before advancing.
